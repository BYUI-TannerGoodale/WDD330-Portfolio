This is an API driven SPA simulation. The app uses the Fetch API and the fakeapi API to simulate calls to an API server. LocalStorage is used as the acting database. The app preforms GET, POST and PUT operations using Fetch. To start, enter the username 'OwnerAdmin@work.co' and the password 'Password'.

If you look in the console, you will see an array of 'users'. The username will be stored in any one of these user's profile. The password is 'Password' for all of them. Different users see a different view and have different interactions. Feel free to browse, make edits, additions and explore the different views available.
